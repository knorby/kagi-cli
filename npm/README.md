# kagi-cli

This package installs the native `kagi` CLI binary for the current platform.

Global install examples:

```bash
npm install -g kagi-cli
pnpm add -g kagi-cli
bun add -g kagi-cli
```

Then run:

```bash
kagi --help
```

The package name is `kagi-cli`, but the installed command is always `kagi`.
